PawPedia is a desktop pet management application that provides pet owners with a centralised platform to manage their pets' information and schedule important reminders.  Built with JavaFX, the application offers an intuitive interface for tracking multiple pets, organising pet-related tasks, and never missing important appointments or activities.

## Project Context

This project was developed for the unit CAB302 (Agile Software Engineering) at QUT. This project aimed to demonstrate an understanding of the following concepts:
- Object-oriented programming principles
- MVC architectural pattern
- Database integration
- GUI development with JavaFX
- Software testing practices
- Version control with Git

## Technology Stack

**Frontend:**
- **JavaFX 21** - Modern desktop UI framework
- **FXML** - Declarative UI markup
- **CSS** - Custom styling

**Backend:**
- **Java 17** - Core application logic
- **SQLite** - Local database for data persistence
- **JDBC** - Database connectivity

**Architecture:**
- **MVC Pattern** - Model-View-Controller architecture
- **DAO Pattern** - Data Access Object for database operations
- **Singleton Pattern** - Database connection management

**Build & Testing:**
- **Maven** - Project management and dependency resolution
- **JUnit 5.10.0** - Unit testing framework
