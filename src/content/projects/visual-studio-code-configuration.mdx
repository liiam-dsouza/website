This is my meticulously crafted Visual Studio Code configuration repository that provides a reproducible, minimalist editor setup.  This project serves as a complete snapshot of my personal VS Code environment, enabling quick replication across machines and serving as documentation for my development workflow.

My configuration emphasises a distraction-free coding experience with a carefully curated aesthetic built around the Catppuccin Mocha theme, relative line numbers for efficient navigation, and extensive UI customisations that strip away visual clutter while maintaining functionality.

## Project Context

Modern developers often work across multiple machines - personal computers, work laptops, remote environments, and maintaining a consistent editor configuration can be challenging. This project solves that problem by version-controlling the entire VS Code setup, making it portable and shareable.

The repository was designed with several goals in mind:
- **Portability**: Easily replicate the exact editor environment on any machine
- **Documentation**: Serve as a reference for configuration choices and customizations
- **Minimalism**:  Create a focused coding environment by removing unnecessary UI elements
- **Aesthetics**: Maintain a cohesive visual experience through theme and icon consistency

The configuration reflects a philosophy of intentional design - every setting has been deliberately chosen to either improve productivity or enhance the visual experience.  The hidden status bar, concealed activity bar, and disabled minimap are all conscious choices to maximize screen real estate for what matters most: **the code**.

## Key Features

### Comprehensive Configuration Files
- **settings.json**: Over 190 lines of extensively commented VS Code settings covering theme, appearance, editor behavior, Git integration, UI customisations, and more
- **extensions. txt**: Curated list of essential extensions including the Catppuccin theme, icon pack, and custom UI styling tools

### Minimalist UI Approach
- Hidden activity bar, status bar, and minimap for maximum screen real estate
- Disabled breadcrumbs, sticky scroll, and CodeLens for reduced visual clutter
- Hidden scrollbars while maintaining scrolling functionality
- Transparent overview ruler markers to eliminate distracting indicators
- Single tab mode and right-aligned sidebar for a unique layout

### Advanced Editor Customizations
- Relative line numbers for efficient Vim-style navigation
- JetBrains Mono Nerd Font with ligatures enabled for improved readability
- Increased line height (2x) for better visual breathing room
- Italic rendering for code comments
- Custom traffic light positioning on macOS for aesthetic appeal
- Extensively styled quick input widget and find dialog

### Developer Experience Enhancements
- Automatic file saving after delays to prevent lost work
- Trailing whitespace trimming on save for cleaner commits
- IntelliSense and parameter hints configured for optimal suggestions
- Emmet expansion support for rapid HTML/CSS development
- Disabled Git decorations for a cleaner explorer view

### Installed Extensions

The configuration uses three carefully selected extensions:

**Catppuccin Theme** (<Highlight>catppuccin.catppuccin-vsc</Highlight>)
**Purpose:** Provides the Catppuccin color theme for VS Code

This extension applies the "Catppuccin Mocha" theme, which is a warm, pastel-based color scheme designed to be easy on the eyes during long coding sessions.  The Mocha variant specifically uses darker backgrounds with soothing purple, pink, and blue accent colors.

**Catppuccin Icons** (<Highlight>catppuccin.catppuccin-vsc-icons</Highlight>)
**Purpose:** Provides matching file and folder icons

This extension complements the Catppuccin theme by adding consistent iconography throughout the editor. The icons use the same color palette as the theme, creating a cohesive visual experience.  The configuration specifically hides explorer arrows for an even cleaner look.

**Custom UI Style** (<Highlight>subframe7536.custom-ui-style</Highlight>)
**Purpose:** Advanced UI customization through custom CSS

This extension allows deep customization of VS Code's interface through custom stylesheets and Electron settings. It enables modifications that aren't possible through standard VS Code settings, such as:
- Custom positioning of macOS traffic lights (window control buttons)
- Removal of box shadows and borders
- Custom positioning of UI elements
- Fine-tuned spacing and padding adjustments

### Notes

- This configuration is optimized for **focus and minimalism** over displaying all possible information
- Many features are hidden but still accessible via keyboard shortcuts
- The setup works best on larger screens where the zoom level and spacing are comfortable
- Some customisations may require VS Code restart to take full effect
- The configuration disables automatic updates (`"update.mode": "none"`) - remember to update manually
